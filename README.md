# Scale Up Pro

Your new low cost super employee - Complete business command center with marketing, CRM, invoicing, and AI support.

## Progressive Web App (PWA)

Scale Up Pro is built as a Progressive Web App that works on iOS and Android as a standalone application.

### Installation

#### iOS (iPhone/iPad)
1. Open Safari and navigate to the Scale Up Pro website
2. Tap the Share button (square with arrow pointing up)
3. Scroll down and tap "Add to Home Screen"
4. Tap "Add" in the top right corner
5. Scale Up Pro will now appear as an app icon on your home screen

#### Android
1. Open Chrome and navigate to the Scale Up Pro website
2. Tap the three-dot menu in the top right
3. Tap "Install app" or "Add to Home Screen"
4. Follow the prompts to install
5. Scale Up Pro will now appear in your app drawer

### Features
- Works offline with cached content
- Full-screen standalone app experience
- Push notifications support
- Fast loading with service worker caching
- Native-like interface on mobile devices

## Development

To run locally:
1. Serve the files using any static file server
2. Access via `http://localhost` or your server address
3. The PWA will work on mobile devices when served over HTTPS
